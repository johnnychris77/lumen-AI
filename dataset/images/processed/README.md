# images/processed/

EXIF-stripped, de-identified images that have passed image-quality grading
(`Excellent`/`Good`/`Acceptable`; see `docs/lcid/IMAGE_QUALITY_GUIDE.md`)
and are ready to enter the annotation/review workflow
(`docs/lcid/ANNOTATION_GUIDE.md`). Not yet Ground Truth — that only happens
after Primary Review + independent Second Review agreement or clinical
adjudication (`docs/lcid/GROUND_TRUTH_GUIDE.md`).
