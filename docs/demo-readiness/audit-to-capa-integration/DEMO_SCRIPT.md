# LumenAI Audit-to-CAPA Integration Demo Script

## Demo Title
LumenAI Audit-to-CAPA Integration: From Governance Signal to Corrective Action

## Demo Objective
Demonstrate how LumenAI connects Enterprise Audit Command Center visibility to CAPA Workflow execution, creating a traceable pathway from audit signal detection to corrective and preventive action.

## Opening Statement
This integration shows how LumenAI moves from audit visibility into accountable quality action. The Enterprise Audit Command Center identifies governance and audit signals, while the CAPA Workflow converts those signals into structured corrective and preventive action records.

## Demo Flow

### 1. Start at the Main LumenAI App
Open:

https://lumen-ai-1.onrender.com

Show the main dashboard and locate the Audit-to-CAPA Integration governance bridge card.

### 2. Explain the Governance Bridge
Point out the section:

Audit-to-CAPA Integration · Governance Bridge

Explain:
This card connects the Audit Command Center and CAPA Workflow into one leadership-facing governance pathway.

### 3. Review Integration Metrics
Show:
- Audit Events
- High-Value Events
- Open CAPAs
- High-Risk CAPAs

Explain:
These metrics show the relationship between enterprise audit visibility and CAPA follow-up.

### 4. Walk Through the Workflow Pathway
Review the pathway:
- Audit signal detected
- High-value event identified
- CAPA review triggered
- Owner assigned
- Corrective action defined
- Preventive action defined
- Governance summary available

Explain:
This creates a traceable chain from quality signal to operational accountability.

### 5. Validate the Integration Endpoint
Open or reference:

https://lumen-ai-53u4.onrender.com/api/enterprise/audit-to-capa/summary

Expected:
- status: success
- module: audit_to_capa_integration
- version: 1.0.0
- Audit Command Center status: healthy
- CAPA Workflow status: healthy

### 6. Reference Supporting Modules

### 7. Closing Statement
The Audit-to-CAPA Integration demonstrates that LumenAI is not just detecting issues. It is creating an enterprise quality intelligence layer that connects audit signals, governance visibility, CAPA execution, and executive oversight.

## Final Demo Message
LumenAI connects frontline quality evidence to enterprise governance and accountable corrective action.
