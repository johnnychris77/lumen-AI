# LumenAI Investor Technical README

## What LumenAI Is

LumenAI is an evidence governance platform for sterile processing, vendor tray accountability, quality events, and infection-prevention-ready documentation.

## Problem

Hospitals struggle to prove whether instrument conditions are acceptable, whether vendor tray issues were reviewed, whether SPD / OR / IP / vendor stakeholders aligned, and whether evidence packets are complete and unchanged.

## Solution

Quality event -> evidence capture -> vendor baseline comparison -> hospital approval -> persistent audit trail -> governance packet -> PDF export -> packet hash -> hash verification.

## Differentiators

- SPD / OR / IP domain workflow
- Vendor baseline governance
- Persistent audit trails
- Governance PDF packet export
- Export history tracking
- SHA-256 tamper-evident packet hashing
- Packet hash verification
- Frontend verification and certificate UI

## Current Stage

Enterprise MVP / early platform prototype.

Ready for:
- Internal demo
- Investor narrative refinement
- Pilot discovery
- Architecture hardening
- Security roadmap development

Not yet ready for unrestricted enterprise production until authentication, RBAC, tenant isolation, tests, and security controls are hardened.
