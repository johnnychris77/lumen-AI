# LumenAI SOC 2 and HIPAA Readiness Map

## Purpose

This document maps LumenAI controls to enterprise compliance readiness expectations. LumenAI is not yet SOC 2 or HIPAA certified.

## SOC 2 Security

Current:
- Audit trail concepts
- Export tracking
- Packet hashing

Needed:
- Real authentication
- RBAC
- Tenant isolation
- Secrets management
- Vulnerability scanning
- Access reviews
- Incident response

## SOC 2 Availability

Needed:
- Uptime monitoring
- Health checks
- Backups
- Disaster recovery
- Error monitoring

## SOC 2 Processing Integrity

Needed:
- Automated tests
- Data validation
- Migration checks
- Reconciliation reports

## SOC 2 Confidentiality

Needed:
- Encryption at rest
- Secure file storage
- Role-based access
- Retention controls

## HIPAA Readiness

Potential PHI may exist in evidence images, notes, or metadata. Required controls include role-based access, audit logging, encryption, data minimization, access review, and BAA readiness.

## Readiness Target

Current: MVP compliance workflow readiness.

Target: SOC 2 / HIPAA pre-assessment readiness.
