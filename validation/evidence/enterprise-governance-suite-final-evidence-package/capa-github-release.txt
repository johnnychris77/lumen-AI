# LumenAI CAPA Governance Scorecard Release Notes v1

## Release Status
RELEASED

## Release Version
v1.0.0

## Release Date
2026-05-30

## Product Area
CAPA Governance, Quality Workflow, and Executive Reporting

## Release Summary
The LumenAI CAPA Governance Scorecard v1 expands the CAPA Workflow from basic corrective and preventive action tracking into an executive governance layer.

This release adds persistent CAPA storage architecture, status updates, overdue escalation monitoring, Power BI-ready export, governance scorecard metrics, frontend scorecard visibility, and portfolio page updates.

---

# 1. Released Capabilities

## 1.1 Persistent CAPA Database

### Status
Released

### Capability
CAPA records are now supported through SQLite-backed persistence architecture instead of relying only on an in-memory store.

### Business Value
Improves CAPA workflow continuity and prepares the module for longer-term database-backed production storage.

---

## 1.2 CAPA Status Update Workflow

### Status
Released

### Capability
CAPA records can be updated across workflow states and ownership/action fields.

### Supported Fields
- status
- owner
- due_date
- risk_level
- description
- corrective_action
- preventive_action

### Supported Workflow States
- open
- in_progress
- pending_review
- closed

### Production Endpoints
- GET /api/capa/{capa_id}
- PATCH /api/capa/{capa_id}

---

## 1.3 CAPA Overdue Escalation

### Status
Released

### Capability
The CAPA workflow can identify overdue CAPAs, due-soon CAPAs, high-risk overdue CAPAs, and CAPAs requiring escalation.

### Production Endpoint
GET /api/capa/escalation-summary?days_until_due=7

### Validated Outputs
- open_capas
- overdue
- due_soon
- high_risk_overdue
- requires_escalation
- overdue record list
- due-soon record list
- high-risk overdue record list

### Business Value
Supports leadership review of time-sensitive CAPAs and escalation readiness.

---

## 1.4 CAPA Power BI Export

### Status
Released

### Capability
The CAPA workflow now supports a Power BI-ready CSV export for dashboards, analytics, governance scorecards, and executive reporting.

### Production Endpoint
GET /api/capa/powerbi-csv?limit=500

### Validated CSV Fields
- capa_id
- title
- source
- risk_level
- status
- owner
- due_date
- created_at
- updated_at
- days_to_due
- is_overdue
- is_high_risk
- is_open
- corrective_action
- preventive_action
- description

### Frontend Integration
The CAPA Workflow Panel includes a visible button:

Download Power BI CSV

## 1.5 CAPA Governance Scorecard

### Status
Released

### Capability
The CAPA Governance Scorecard provides an executive view of CAPA performance and governance risk.

### Production Endpoint
GET /api/capa/governance-scorecard?days_until_due=7

### Scorecard Metrics
- total_capas
- open_capas
- closed_capas
- high_risk_capas
- overdue_capas
- due_soon_capas
- high_risk_overdue_capas
- requires_escalation
- closure_rate_percent
- powerbi_export_ready

### Governance Status Values
- healthy
- watch
- action_required

### Business Value
Provides leadership with a concise view of CAPA performance, escalation risk, overdue exposure, closure rate, and analytics readiness.
