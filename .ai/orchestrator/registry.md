# LumenAI Agent Registry

## Available Agents

### architect
Owns architecture, issue shaping, boundaries, refactor logic, and cross-service consistency.

### backend
Owns API, worker, DB, queue, service layer, and report generation interfaces.

### frontend
Owns UI flows, upload experience, polling, history, and report navigation.

### ml
Owns inference logic, evaluation discipline, model metadata, and output schema evolution.

### devops
Owns Docker, Compose, CI/CD, GHCR, staging, runtime config, and observability.

### qa-release
Owns validation, regression checks, release risk notes, and readiness summaries.
