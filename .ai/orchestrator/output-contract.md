# LumenAI Output Contract

Every agent execution must produce:

## 1. Summary
What changed and why.

## 2. Files Touched
List of repo paths affected.

## 3. Validation
Commands run and observed result.

## 4. Risks
Known risks, assumptions, or follow-ups.

## 5. Release Impact
One of:
- none
- patch
- minor
- infra
- runtime-sensitive

## 6. Recommended Next Step
Single best next action.
